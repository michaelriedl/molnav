Welcome to MolNav's documentation!
===================================

**MolNav** (Molecular Navigator) is a comprehensive Python library for navigating the future of molecular modeling. 
It implements state-of-the-art deep learning models for molecular modeling using PyTorch.

.. note::
   This project is under active development.

Features
--------

* State-of-the-art transformer models for molecular modeling
* PyTorch-based implementations
* Easy-to-use API
* Comprehensive documentation and examples

Installation
------------

To install MolNav from source:

.. code-block:: bash

   git clone https://github.com/michaelriedl/molnav.git
   cd molnav
   uv sync --all-extras

Quick Start
-----------

Here's a simple example to get you started:
